# VIA-KORE - Portal de Inteligência

Portal de Business Intelligence da Via Group, desenvolvido com Next.js e FastAPI.

## 🚀 Tecnologias

- **Frontend**: Next.js 14, React, TypeScript, TailwindCSS
- **Backend**: FastAPI (Python), MySQL
- **Autenticação**: NextAuth.js
- **BI**: Power BI Embedded

## 📋 Funcionalidades

- ✅ Autenticação de usuários
- ✅ Dashboard de relatórios Power BI
- ✅ Gerenciamento de usuários (CRUD)
- ✅ Gerenciamento de dashboards
- ✅ Gerenciamento de credenciais Power BI
- ✅ Interface responsiva e moderna
- ✅ Controle de acesso por função (Admin/User)
